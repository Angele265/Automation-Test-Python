# Automation Exercise Selenium Framework

## Description
Selenium Python automation framework for testing the Automation Exercise website.

## Technologies
- Python 3.12
- Selenium WebDriver
- Pytest
- WebDriver Manager

## Installation

Clone repository:

git clone https://github.com/Angele265/Automation-Test-Python.git

Install dependencies:

pip install -r requirements.txt

## Run Tests

pytest

## Project Structure

pages/
tests/
utils/

## Test Cases

- Verify Home Page
- Register New User
- Login User
